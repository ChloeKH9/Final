Hangman

Description:

Hangman is a simple game where letters are guessed to form a word. This code has a list of shows that are selected at random. First, blank bars are printed in a list to match the chosen word. The user is able to input a letter which the program then checks if it’s in the word. If it is, it’ll loop through the word, find where the letter is, and set the bar equal to the guess. There’s also two lists, one for correct guesses and one for incorrect ones, that will contain letters as the player continues to guess. There are 14 chances, so if the user does not guess the whole thing by the time the chances hit zero, the player loses.
