Page 3: Admission 🎼

Code for admission prices

Description:

I plotted three different variables on a line graph: prices of musicals, plays and special tickets. The years range from 2007 to 2018 and they act as the x-axis. Afterwards, the user can input a year and type to call specific information. However, I didn’t put a system to correct if the user inputs something out of the list or spells it wrong, so it’ll simply return an error if that occurs.
